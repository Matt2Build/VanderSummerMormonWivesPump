# VanderSummerMormonWivesPump

Reality TV drama database for the obsessed.

## Features

- **Cast Tracker** — Cross-show relationship web
- **Receipt Vault** — Searchable drama archive
- **Drama Score** — Cast sentiment & trends
- **Prediction Pools** — Yes/no bets with leaderboard

## Stack

- Next.js 14
- Supabase
- Tailwind CSS
- Vercel
